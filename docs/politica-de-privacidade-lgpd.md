# Política de Privacidade e LGPD — IMOBIPRO (versão base)

**Última atualização:** 27/12/2025

## 1. Dados coletados
- Dados de conta: nome, e-mail, identificadores de autenticação.
- Dados operacionais: imóveis, leads, clientes, agenda, tarefas, anotações e históricos.
- Dados de cobrança: status de assinatura/pagamentos (sem armazenar dados completos de cartão).

## 2. Finalidade
- Prestação do serviço contratado.
- Segurança, prevenção a fraudes e auditoria.
- Comunicação sobre o serviço (trial, cobranças, avisos).

## 3. Base legal
- Execução de contrato.
- Legítimo interesse (segurança e melhoria do serviço).
- Cumprimento de obrigações legais/regulatórias, quando aplicável.

## 4. Compartilhamento
- Provedores essenciais: Supabase (infra de dados), Vercel (hospedagem), Mercado Pago (pagamentos), Resend (e-mail), WhatsApp Cloud API (mensagens).
- Não vendemos dados pessoais.

## 5. Direitos do titular
Você pode solicitar: confirmação, acesso, correção, anonimização, portabilidade, eliminação e informações sobre compartilhamento, nos termos da LGPD.

## 6. Segurança
Empregamos controles de acesso, regras por perfil (RLS), logs e backups. Nenhuma medida é infalível; comprometemo-nos a adotar boas práticas.

## 7. Retenção
Mantemos os dados pelo tempo necessário à prestação do serviço e cumprimento de obrigações legais. Após cancelamento, podemos reter dados por período mínimo para auditoria e recuperação, quando justificado.
